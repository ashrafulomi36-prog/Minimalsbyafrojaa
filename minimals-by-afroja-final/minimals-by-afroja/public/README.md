# Public Assets

Place the following files in this directory:

## Required
- `og-image.jpg` (1200×630px) — Open Graph image used for social sharing
  - Design it with your brand colors (#E8B4B8, #8B5E3C) and logo
  - Tools: Canva, Figma, or Adobe Express

## Optional (for offline fallback)
- `favicon.ico` — Browser tab icon
- `apple-touch-icon.png` (180×180px) — iOS home screen icon
- `icon-192.png` / `icon-512.png` — PWA icons

## Notes
- Product images are served from Sanity CDN (not stored here)
- All images are optimized automatically via Next.js Image component
